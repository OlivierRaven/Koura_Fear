# Fear of the new: a predatory invader may provoke stronger non-consumptive effects on a keystone native prey than a native predator

Olivier V. Raven | olivier.raven@icloud.com

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19716761.svg)](https://doi.org/10.5281/zenodo.19716761)

## Overview

This repository contains all data, analysis code, and manuscript
files for a laboratory experiment testing whether the invasive brown
bullhead catfish (*Ameiurus nebulosus*) elicits stronger
non-consumptive effects (NCEs) on kōura (*Paranephrops planifrons*)
than the native longfin eel (*Anguilla dieffenbachii*). Refuge use
and feeding rate were quantified under predator cue exposure across
light and dark conditions.

**Key findings:**

- Catfish cues significantly increased kōura refuge use; eel cues did
  not, despite eels being a natural predator
- Kōura were more likely to use refuge in the dark than in light
  conditions
- Refuge use increased after predator exposure relative to before,
  suggesting a carry-over fear effect
- Feeding rate was not significantly affected by predator treatment

## Reproducing the analysis

This project uses [Quarto](https://quarto.org/) and R.
All analyses are contained in `analysis.qmd`.

### Requirements

- R >= 4.4
- Quarto >= 1.4
- R packages: `glmmTMB`, `emmeans`, `DHARMa`, `AICcmodavg`,
  `multcomp`, `tidyverse`, `patchwork`, `ggpattern`, `DT`, `car`

### Install R packages

    install.packages(c("glmmTMB", "emmeans", "DHARMa", "AICcmodavg",
                       "multcomp", "tidyverse", "patchwork", "ggpattern",
                       "DT", "car", "lme4", "lmerTest", "readxl"))

### Render the analysis

    quarto render analysis.qmd

The rendered HTML analysis is available at:
https://olivierraven.github.io/NCE_kōura_catfish/
